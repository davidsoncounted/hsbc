Setting up a new Secure Key

How do I set up a new Secure Key? 

Digital Secure Key - You'll need to have already registered for digital banking and chosen a Digital Secure Key.

  1. Open the HSBC UK Mobile Banking app on your device
  2. Enter your username
  3. Select an option to verify it's you - either by entering your password (in full, not just 3 characters) and an activation code sent by text message or, if you already have a Secure Key on another device, via a QR code from one of your other devices
  4. Follow the steps to complete the chosen action
  5. Create and confirm your new Digital Secure Key PIN
  6. You can now log on to the HSBC UK Mobile Banking app using the Digital Secure Key PIN you've just created


Physical Secure Key - You'll need to have received your physical Secure Key (this could take 10 working days from the time you've requested).

  1. Select 'Log on' via hsbc.co.uk
  2. Enter your username and then password (in full, not just 3 characters)
  3. If you have your Secure Key with you, select the 'Activate now' option
  4. Enter the serial number, which is on the back of your Secure Key
  5. Follow the on-screen steps to create your Secure Key PIN
  6. Enter the activation code you received in the letter that came with the Secure Key. If you don't have this or it's expired, there's an option to generate a new activation code on-screen
  7. Follow the instructions to generate a security code on your Secure Key and enter it in the required field
  8. Select 'Continue' and you'll see confirmation that your Secure Key has been set up successfully

How do I set up the mobile banking app on a new device? 

You can set up app access on up to 3 different devices. To do this:

  1. Open the HSBC UK Mobile Banking app.
  2. Enter your username.
  3. Select an option to verify it's you. This can be done by entering your password (in full) and a one-time code sent by text message, or through a QR code from one of your other devices (if you have one).
  4. Follow the steps to complete the chosen action.
  5. Create and confirm your new Digital Secure Key PIN.

You can now log on to the HSBC UK Mobile Banking app using the Digital Secure Key PIN you've just created. 

If you have access to the app through a device you no longer use, you should remove it via 'Manage devices'. To do this, go to the Profile section of the app and select 'Security and generate security code'. If you're using online banking, select 'Account services' in the top menu to find 'Manage devices'.

Alternatively, you can contact us on 03456 002 290 so we can remove the device for you. This will prevent fraudulent access to your accounts.

How do I set up a replacement Secure Key when the battery is low on my existing one? 

If you can still use your Secure Key, but would like to replace it, you can do either of the following:

  • upgrade to a Digital Secure Key by downloading the HSBC UK Mobile Banking app
  • order a new physical one using Chat

Just log on to online banking and select 'Chat'.

To upgrade to a Digital Secure Key:

  1. Open the HSBC UK Mobile Banking app.
  2. Enter your username.
  3. Verify it's you by entering your security code.
  4. Follow the steps to complete the chosen action.
  5. Create and confirm your new Digital Secure Key PIN.

If you've chosen to receive a new physical Secure Key, you'll need to wait to receive it and then follow these steps:

  1. Log on to online banking.
  2. Enter your username and then security code.
  3. If you have your Secure Key with you, select 'Activate now'.
  4. Enter the serial number, which is on the back of your new Secure Key.
  5. Follow the on-screen steps to create your Secure Key PIN.
  6. Enter the activation code if you've already received it. If you don't have this or it's expired, there's an option to generate a new activation code on screen.
  7. Follow the instructions to generate a security code on your existing Secure Key and enter it in the required field.
  8. Repeat the instructions to generate a security code from your new Secure Key and enter it in the required field.
  9. Select 'Continue' and you'll see confirmation that your Secure Key has been set up successfully.


You'll have 30 days to activate your new Secure Key. After this time, you won’t be able to log on with your existing Secure Key and will need to activate your new one.

 

Logging on with your Secure Key

How do I log on to online banking using my Digital Secure Key? 

  1. Launch the HSBC UK Mobile Banking app and select 'Generate security code'
  2. Select 'Log on security code' from the list
  3. Enter your Digital Secure Key PIN and select 'Generate code'. If you have face or fingerprint recognition set-up, this can also be used to generate a code
  4. Your log on security code will be displayed
 

How do I generate a one-time passcode using my physical Secure Key?  

To generate a ‘log on security code’ to log on to online banking with your physical Secure Key, follow these instructions:

  1. Press and hold the green button to turn on your Secure Key, then enter your Secure Key PIN.
  2. With the HSBC welcome screen displayed, press the green button to generate a security code.
  3. The security code will be shown on your Secure Key screen.

Resetting your log on details

What if I've forgotten or locked my Secure Key PIN?  

Digital Secure Key:

If you’ve forgotten your Digital Secure Key PIN, open the HSBC UK Mobile Banking app, select the ‘Forgotten Digital Secure Key PIN’ link and follow the on-screen instructions to set up a new PIN.

If you've locked your Digital Secure Key, open the HSBC UK Mobile Banking app, select the 'Reset Digital Secure Key PIN' option and follow the on-screen instructions to reset.

Physical Secure Key:

If you enter an incorrect PIN, you'll see the message 'FAIL 1'. Press the green button to enter your PIN again. You'll have 3 attempts to enter your PIN correctly. After the 3rd incorrect attempt ('FAIL 3'), the Secure Key will become locked and will display a 7-digit code on screen whenever it's turned on.

To continue using online banking, you’ll need to reset your PIN. To do this, go to our online banking log on page, enter your username, select the '?' icon and then ‘Forgot your Secure Key PIN’. Follow the on-screen instructions to reset your PIN.

What if I've forgotten my password?  

Go to the HSBC UK Mobile app or the online banking log on page and enter your username, then select ‘forgot your password’ and follow the instructions.

To complete your new password set-up, you’ll be asked to call our automated phone service on 0345 600 2290. We'll use your telephone security number or voice ID to verify you, then when you're asked what you'd like to do, say 'reset password'. Alternatively, you can wait for the options and select option 2.

If you can access online banking, you can also log on there and select 'Change your password' from the 'Account services' menu.

What if I've forgotten my username? 

If you've forgotten your username, you’ll need to open your mobile app or visit the online banking log on page. Select ‘Forgot your username?’ and follow the on-screen instructions. 

Note: You'll need to bring your sort code and account number, or long card number if you have a credit card.

What if I'm locked out of the app or online banking?  

Your next steps depend on which security details you've locked out or forgotten. To make sure you follow the right steps, go to our digital banking help tool and select the most appropriate option from the ones presented.

Can I change my security details?  

If you want to change your username or password, you can do this when logged on to online banking. Select 'Account services' in the top menu, then ‘Change personal security details’. This is where you can amend your existing security details.

To change your Digital Secure Key PIN, log on to your mobile app, go to the Profile section and select 'Security and generate security code'.

If you've forgotten your Digital Secure Key PIN, open the app, select the 'Forgotten Digital Secure Key PIN' link and follow the on-screen instructions.

To change your physical Secure Key PIN, go to the online banking log on page and enter your username. Then, select 'Forgot your Secure Key PIN?' and follow the on-screen instructions to change it.

Switching or replacing your Secure Key

How do I set up the app on a new device? 

If you no longer have the old device with your app on it, you should first deactivate that old device. If you have another device with the app already, you can do this there. To view or remove devices, select the profile icon at the top of the app screen, then the 'Security' option and 'Manage my devices'. If you can't do this, please contact us on 03456 002 290 so we can deactivate the old device for you. This will prevent fraudulent access to your accounts. 

Once you have a new device, the steps to set up the app are:

1. Open the HSBC UK Mobile Banking app

2. Enter your username

3. Select an option to verify it's you - either by entering your password (in full, not just 3 characters) and a one-time code sent by text message, or via a QR code if you have the app on another device that you can still access

4. Follow the steps to complete the chosen action

5. Create and confirm your new Digital Secure Key PIN

6. You can now log on to the HSBC UK Mobile Banking app using the Digital Secure Key PIN you've just created

Note: Once you have set up your digital secure key pin, you can set up access to the app on up to 3 different devices. Simply follow the above steps on each device.

What happens if I lose a device with my app on it?  

If you no longer have the device with your app on it, you should remove it in 'Manage devices'. You can do this through another device by going to the Profile section of the app and selecting 'Security and generate security code'. 

If you can't do this, you can contact us on 03456 002 290 so we can remove the device for you. This will prevent fraudulent access to your accounts.

Once you have a new device, you can set up the app:

  1. Open the HSBC UK Mobile Banking app.
  2. Enter your username.
  3. Select an option to verify it's you. This can be done by entering your password (in full, not just 3 characters) and a one-time code sent by text message, or through a QR code from one of your other devices (if you have one).
  4. Follow the steps to complete the chosen action.
  5. Create and confirm your new Digital Secure Key PIN.

You can now log on to the HSBC UK Mobile Banking app using the Digital Secure Key PIN you've just created. 

Note: Once you have set up your Digital Secure Key PIN, you can set up app access on up to 3 different devices. To do this, follow the above steps on each device.

What happens if I lose or break my physical Secure Key? 

If it’s broken but you can still manage to log on, you can order a new physical Secure Key using Live Chat. First, log on to online banking, then look for the Chat tab. 

You can also contact us on 03456 002 290 so we can replace the Secure Key. You can either upgrade to a Digital Secure Key or arrange for a replacement physical Secure Key to be sent to you.

Please note that you need a Digital Secure Key if you want to log on to our mobile banking app. If you currently have a physical Secure Key and would like access to the app, please switch to a Digital Secure Key instead. 

How do I switch from a physical Secure Key to a Digital Secure Key?  

If your physical Secure Key is still working:

  1. Open the HSBC UK Mobile Banking app. You'll need to download it if you don't have it.
  2. Enter your username.
  3. Select an option to verify it's you. Do this either by entering your password (in full, not just 3 characters) and a one-time code sent by text message, or via a code generated on your physical Secure Key.
  4. Follow the steps to complete the chosen action.
  5. Create and confirm your new Digital Secure Key PIN.

If your physical Secure Key is lost or broken:

  1. Call us on 03456 002 290 so we can deactivate your physical Secure Key.
  2. Open the HSBC UK Mobile Banking app. You'll need to download it if you don't have it.
  3. Enter your username.
  4. Enter your password (in full, not just 3 characters) and a one-time code sent by text message.
  5. Follow the steps to complete the chosen action.
  6. Create and confirm your new Digital Secure Key PIN.


And that's it. You'll now be able to log on using your Digital Secure Key.

You'll no longer need your physical Secure Key. You can recycle it by handing it in to your nearest HSBC UK branch or disposing of it at your local recycling centre. Please don’t put an old Secure Key into your normal household waste bin.

How long will it take for a replacement Secure Key to arrive? 

Please allow up to 10 working days to receive the physical Secure Key in the post.

If your Secure Key is lost, damaged or stolen you won’t be able to access online banking until you receive and activate your new Secure Key. If your Secure Key has a low battery, you can still use this to access online banking for 30 days until you have to activate your new one.

Remember, you can always switch to a Digital Secure Key. This will get you back online straight away and also allow you to use the HSBC UK Mobile Banking app. Our app lets you manage your accounts easily and securely from a time and a place that suits you. Its unique features include cheque deposit, instant transaction alerts and Balance After Bills.

To switch to a Digital Secure Key, see 'How do I set up a replacement Secure Key when the battery is low on my existing one?'.

How long does the battery last on a physical Secure Key?  

It should last 5 years with average use. The device will warn you when the battery is low. At that point, you can either upgrade to a Digital Secure Key or you can contact us to order a replacement. It would take up to 10 working days for a replacement to arrive.

If you want to make sure your access isn't affected, the best option is to upgrade to a Digital Secure Key. To do this:

  1. Open the HSBC UK Mobile Banking app. You'll need to download it if you don't have it.
  2. Enter your username.
  3. Select an option to verify it's you - either by entering your password (in full, not just 3 characters) and a one-time code sent by text message, or via a code generated on your physical Secure Key.
  4. Follow the steps to complete the chosen action.
  5. Create and confirm your new Digital Secure Key PIN.

Can I recycle my old physical Secure Key?  

Please dispose of an old or broken physical Secure Key in an environmentally friendly manner. You can do this by dropping it off at any HSBC branch or disposing of it at your local recycling centre. Please don’t put an old Secure Key into your normal household waste bin.

Around 60% of the components of our physical Secure Keys are recyclable. We arrange for all physical Secure Keys returned to us to be dismantled and recycled. All non-recyclable parts will be handled in accordance with EU regulations.

What options do you have for customers with accessibility requirements?  

The Digital Secure Key is compatible with normal VoiceOver (iOS) or TalkBack (Android) features. If you can't use the Digital Secure Key, and you'd like to discuss other Secure Key options, contact us on 0345 600 2290.

Managing your devices

How do I view which devices I have the app on? 

If you're using the app, go to the Profile section and select 'Security and generate security code'. Then, select 'Manage devices' to see the full list of devices you've set up.

If you're using online banking, select 'Account services' in the top menu. From there, choose 'Manage Secure Key'.

How do I remove a device? 

You can remove a device through either the app or online banking.

If you're using the app, go to the Profile section and select 'Security and generate security code'. Then, select 'Manage devices' to see the full list of devices you've set up. You'll be able to remove your devices here. 

If you're using online banking, select 'Account services' in the top menu. From there, choose 'Manage Secure Key'. You'll see a full list of devices that you've set up. Under each device, there is an option to 'Remove device'.

Note: If the device being removed is your only device, then you'll need to activate a new device before you can use digital banking again.

How do I set up the app on a new device? 

If you no longer have the old device with your app on it, you should first deactivate that old device. If you have another device with the app already, you can do this there. To view or remove devices, select the profile icon at the top of the app screen, then the 'Security' option and 'Manage my devices'. If you can't do this, please contact us on 03456 002 290 so we can deactivate the old device for you. This will prevent fraudulent access to your accounts. 

Once you have a new device, the steps to set up the app are:

1. Open the HSBC UK Mobile Banking app

2. Enter your username

3. Select an option to verify it's you - either by entering your password (in full, not just 3 characters) and a one-time code sent by text message, or via a QR code if you have the app on another device that you can still access

4. Follow the steps to complete the chosen action

5. Create and confirm your new Digital Secure Key PIN

6. You can now log on to the HSBC UK Mobile Banking app using the Digital Secure Key PIN you've just created

Note: Once you have set up your digital secure key pin, you can set up access to the app on up to 3 different devices. Simply follow the above steps on each device.

Error messages on your physical Secure Key

What does this error message on my physical Secure Key mean? 

To find out what's happening, go to our digital banking help tool, select the option for 'Physical Secure Key error messages' and confirm which error message you're seeing.

Updating your contact details

How do I update my mobile number so I can receive a text message code? 

There are a few ways you can do this.

Through the app:

  1.  Log on to the app.
  2. Select the 'Profile' icon at the top of your screen.
  3. Select the tile with your name in it.
  4. Follow the steps to review and update your contact details.

Through online banking:

  1. Log on via hsbc.co.uk.
  2. Select 'Account services' from the top menu.
  3. Choose the 'Personal and address details' option.
  4. Follow the steps to review and update your contact details.

If you can't access either of the options above, you can call the number on the back of your card or visit a branch to update your details there.
